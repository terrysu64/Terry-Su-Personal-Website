Some Of My Projects

Gamble Bot

The Gamble Bot is a Discord bot that I built last summer using Python and the Discord.py API. The bot itself comprises of various interactive commands and functions that allow users in Discord guilds to gamble among themselves or against the bot using a built-in currency. While running, all guild/currency-related information is tracked live using a set of JSON files. Feel free to invite the Gamble Bot into your own servers as well!

PearPals

PearPals is an entrepreneurial startup that I co-founded with my friends; our service is dedicated to connecting students in order to foster practical yet impactful cross-cultural exchanges! Through using our platform, you will be able to easily network with our users that come from all over the world and learn from their unique backgrounds and experiences!

Recently, using Bubble.io, I migrated the service to a fully-function website. This process entailed the application of a variety of full-stack development concepts such as UI responsiveness, UX design/workflow, and managing back-end databases.

SmartBrain

SmartBrain is a fun full-stack web-application that allows users to create on-site accounts and interact with a machine learning AI capable of detecting human faces in an image; this was implemented using the Clarifai Face Detection API. The front-end pages were built using React.js while back-end server operations were facilitated using Node.js and PostgreSQL.
